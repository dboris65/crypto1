For Galois multiplication, we will use well known formula:

log(x*y) = log(x) + log(y), e.g.
x*y = antilog( log(x) + log(y))

First, we will build exponent and logarithm tables, using one of generators and slow version of algorithm (button GenExpTable).

After that, we can use fast version for Galois multiplication, as follows:

s := ltable[a] + ltable[b];
s := s mod 255;
s := exptable[s];
